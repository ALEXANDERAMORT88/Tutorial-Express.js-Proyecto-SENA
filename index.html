Tutorial de Express desde cero

¿Que es Express?

Express es un framework minimalista para Node.js que permite crear aplicaciones web y APIs de forma sencilla y rápida.

Requisitos previos

Antes de comenzar a trabajar con Express, es importante cumplir con los siguientes requisitos:

Tener instalado Node.js

Node.js es un entorno de ejecución para JavaScript del lado del servidor. Express se ejecuta sobre Node.js, así que es fundamental tenerlo instalado.

¿Cómo instalar Node.js?

  • Ve al sitio oficial: https://nodejs.org/es
  • Descarga la versión recomendada (LTS)
  • Sigue los pasos del instalador según tu sistema operativo (Windows, macOS o Linux)
Nota:

Al instalar Node.js también se instala npm (Node Package Manager), que es esencial para instalar Express y otros paquetes.

Tener conocimientos básicos de JavaScript

Para trabajar con Express necesitas entender conceptos esenciales de JavaScript, como:

  • Variables (let, const)
  • Funciones y funciones flecha
  • Objetos y arreglos
  • Callbacks y promesas
  • Manejo del DOM (opcional para backend, pero útil en general)
  • Sintaxis moderna (ES6+)
